# ahin-page.github.io

GitHub Pages one-page personal site for `https://ahin-page.github.io`.

## Local development

1. `bundle install`
2. `bundle exec jekyll serve`
3. Open `http://127.0.0.1:4000`

## Main files

- `_data/profile.yml`: hero text, links, contact info, photo path
- `_data/resume.yml`: education and experience blocks
- `_layouts/home.html`: one-page homepage layout
- `assets/main.scss`: site styles
- `assets/profile-placeholder.svg`: photo placeholder image
