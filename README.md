# ahin-page.github.io

GitHub Pages blog for `https://ahin-page.github.io`.

## Local development

1. `bundle install`
2. `bundle exec jekyll serve`
3. Open `http://127.0.0.1:4000`

## Publishing

Push to the default branch of this repository and enable GitHub Pages from the repository settings if it is not already enabled.

GitHub Pages will build the site on GitHub. The local Gemfile is only for previewing the blog on your machine.
